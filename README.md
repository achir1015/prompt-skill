# prompt-skill    https://achir1015.github.io/prompt-skill/
以長照為主題、整合 NotebookLM 資料來源，並引導至 Gemini Pro Canvas 的簡報提示詞產生器。
📋 功能架構
左側面板（設定區）

主題輸入：填入您的長照簡報主題
投影片張數滑桿：6～30 張，可自由調整
快速預設：6 種常見情境一鍵套用（家屬衛教、照顧技巧、政策倡議、服務介紹、個案研討、年度報告）
9 個設定維度：類型、角色、邏輯架構、語氣、受眾、加分項目（可複選）、格式、文字量、視覺風格

右側面板（提示詞輸出）

自動附上 NotebookLM 資料源連結（您的筆記本網址）
自動附上 Canvas 操作步驟 → 匯出 Google 簡報 的指引
「複製提示詞」按鈕
「前往 Gemini Pro →」按鈕（直連 https://gemini.google.com/app，有脈動動畫提示）

底部步驟引導（5 步驟清楚標示操作流程）

🔄 使用流程

填入主題 → 選擇設定 → 點「複製提示詞」
點「前往 Gemini Pro →」開新視窗
貼入提示詞 → 送出
在 Gemini 切換 Canvas 模式
匯出 → Google 簡報 ✓
